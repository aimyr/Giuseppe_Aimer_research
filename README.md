# GIuseppe_Aimer_research
Large Language Models (LLMs) are increasingly deployed as evaluators in "LLM-as-a-judge" scenarios. While recent studies indicate that models tend to prefer their own output, it remains unclear whether this preference stems from objective quality improvements or an implicit "self-recognition" bias. This research investigates the causal link between an LLM's ability to detect AI-generated text and its evaluation of that text.
To decouple perceived quality from authorship, we utilize a dataset of human-written debate arguments. We generate parallel versions of these arguments using LLMs at varying levels of intervention: (1) light paraphrasing, (2) stylistic improvement, and (3) complete rewriting, while strictly preserving the original semantic meaning and stance. We then task the models with two objectives: identifying the authorship (AI vs. Human) and scoring the quality of the arguments.
By comparing scores assigned to human originals versus AI-edited variations, we aim to determine if LLMs exhibit an "in-group bias"—systematically favoring text they recognize as synthetic, even when semantic content remains constant. This study addresses a critical safety concern: if LLMs possess a "survival strategy" to blindly favor AI-generated patterns, their utility as neutral evaluators is fundamentally compromised.

<h5>INSTRUCTIONS FOR DATASETS<h5>

This project contains two datasets organized by task structure and evaluation type. The first dataset consists of debate arguments and is designed for argument comparison tasks. The second dataset consists of paraphrased research abstracts and is designed for abstract comparison tasks.

Both datasets are divided into two primary categories: Self Recognition and Ranking/Scoring.

Self Recognition

The self recognition category measures whether a model can identify text that it believes it generated itself. This category contains two subtypes.

Binary self recognition presents the model with a single abstract or argument and asks whether it believes it wrote that text.

Pairwise self recognition presents the model with two candidate texts and asks which of the two it believes it wrote.

Ranking / Scoring

The ranking and scoring category measures how models evaluate text quality. This category contains three subtypes.

Single text scoring asks the model to evaluate one abstract or argument at a time and assign a score, either according to its own judgment or according to a rubric provided by the researcher.

Pairwise comparison asks the model to compare two texts and determine which one is better.

Cross model rubric evaluation first collects evaluation instructions or rubrics generated by different models. Then, each model evaluates individual abstracts or arguments using the rubrics created by the other models.
