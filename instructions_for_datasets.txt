This project contains two datasets organized by task structure and evaluation type. The first dataset consists of debate arguments and is designed for argument comparison tasks. The second dataset consists of paraphrased research abstracts and is designed for abstract comparison tasks.

Both datasets are divided into two primary categories: Self Recognition and Ranking/Scoring.

Self Recognition

The self recognition category measures whether a model can identify text that it believes it generated itself. This category contains two subtypes.

Binary self recognition presents the model with a single abstract or argument and asks whether it believes it wrote that text.

Pairwise self recognition presents the model with two candidate texts and asks which of the two it believes it wrote.

Ranking / Scoring

The ranking and scoring category measures how models evaluate text quality. This category contains three subtypes.

Single text scoring asks the model to evaluate one abstract or argument at a time and assign a score, either according to its own judgment or according to a rubric provided by the researcher.

Pairwise comparison asks the model to compare two texts and determine which one is better.

Cross model rubric evaluation first collects evaluation instructions or rubrics generated by different models. Then, each model evaluates individual abstracts or arguments using the rubrics created by the other models.
